# 기초 명령어

### (1) `pwd`

* pwd(print working directory): 현재 폴더의 경로



### (2) `ls`

* ls



### (3) `cd` [폴더명]

* cd(change directory): 폴더 변경



### (4) `cd..`

* 상위 폴더로 이동



### (5) `mkdir` [폴더명]

* mkdir(make directory): 폴더를 생성

### (6) `touch`

* touch : 파일을 생성



### (7) `echo`

* echo : 화면 출력(python의 )



### (8) `cp` [파일명] [경로명]

* cp(copy) : 파일 (또는 폴더) 복사

* rm -r folder / folder1



### (9) `rm` [파일명]

* rm(remove):  파일 (또는 폴더) 삭제

* rm -r folder/



### (10) `mv` [파일/폴더명]

* mv(move): 파일 (또는 폴더) 이동 & 이름변경
